# CryoAI

CryoAI is an iceberg monitoring software that uses a trained AI model to automatically identify and segment icebergs from satelitte imagery using the U-net algorithm. It is intended to be used by scientists or organizations who regularly track icebergs such as the National US Ice Center. 
The task of tracking icebergs is would usually take hours of tedious work every week, which is where CryoAI can come in and perform this menial task within seconds.

# Technologies 

- Frontend: Typescript
- FastAPI
- Backend: Python
- Model: TensorFlow
- PostgreSQL flexible server
- Cloud for media storage
